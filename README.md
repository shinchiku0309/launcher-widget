# Launcher Widget

Windows 11 向けの軽量ランチャーウィジェットです。C++ と Win32 API だけで実装しており、Electron、Web ランタイム、.NET ランタイムなどの追加実行環境に依存しません。

## 目的

- Stream Deck のようなボタングリッドをデスクトップ上で使えるようにする。
- 常駐しても負担になりにくい、軽量なランチャーにする。
- ウィンドウ、グリッド、ページ、ボタン、アイコン、アクションを GUI で設定できるようにする。
- 将来的なマクロ機能を追加しやすいアクションモデルを保つ。

## 現在の機能

- 常に最前面表示の切り替え。
- ページ切り替え。
- 行数、列数、ボタンサイズ、余白の設定。
- ボタンごとのタイトル、絵文字またはテキスト、アクション種別、実行対象、引数、カスタム画像パスの設定。
- アプリ、ファイル、フォルダー、URL、Windows 設定 URI、コマンド、簡単なキー同時押しの実行。
- `.exe`、`.lnk`、通常のシェル対象からのアイコン抽出。
- GDI+ による `.png`、`.jpg`、`.jpeg`、`.bmp`、`.gif`、`.ico` のカスタム画像表示。
- 設定上は `.svg` パスを受け付けますが、現在の Win32 描画では小型の SVG ラスタライズ処理を内蔵していないため、表示はテキストまたは絵文字にフォールバックします。

## ビルド方法

Visual Studio 2022 で `Launcher.sln` を開き、`x64` / `Release` でビルドしてください。

外部ライブラリの追加インストールは不要です。

## 使い方

- ボタンを左クリック: 設定されたアクションを実行します。
- ボタンを右クリック: そのボタンの設定を編集します。
- 空白部分またはウィンドウ部分を右クリック: メニューを開きます。
- 右上の `SET`: グリッドやウィンドウ設定を編集します。
- 右上の `X`: アプリを終了します。
- `Page +` / `Page -`: ページを切り替えます。
- 右クリックメニューの `Settings`: グリッドやウィンドウ設定を編集します。

## ボタン設定

ボタン編集では、最初にアクション種別を選び、その種別に合った入力欄を設定します。

- `App (.exe)`: `Select` から `.exe` ファイルを選択できます。
- `URL`: URL を直接入力できます。`Favorites` から Windows Favorites、Microsoft Edge、Google Chrome のブックマーク候補を取り込めます。
- `File`: `Select` から任意のファイルを選択できます。
- `Folder`: `Select` からフォルダーを選択できます。
- `Windows Settings`: `ms-settings:` などの Windows 設定 URI を入力します。
- `Command`: コマンドと引数を分けて入力します。
- `Keys`: `CTRL+ALT+T` のようなキー同時押しを入力します。
- `None`: 何もしないボタンにします。

設定ファイルは次の場所に保存されます。

```text
%LOCALAPPDATA%\LauncherWidget\config.ini
```

## アクション種別

- `Open`: `ShellExecute` により、アプリ、ファイル、フォルダー、ショートカット、URL を開きます。
- `Command`: 任意のコマンドラインを、必要に応じて引数付きで起動します。
- `Settings`: `ms-settings:display` などの Windows 設定 URI を開きます。
- `Keys`: `CTRL+ALT+T` のような簡単なキー同時押しを送信します。
- `None`: 何もしません。

## マクロ機能の方向性

現在の実行処理は `ExecuteAction` に集約されています。今後マクロ機能を追加する場合は、既存のアクションレコードのリストと待機時間などのオプションを持つ `Macro` アクション種別を追加する方針です。これにより、実行ロジックの重複を避けながらランタイムを軽量に保てます。
